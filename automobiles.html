Exploratory Data Analysis on the automobile Dataset¶

Report

1. Introduction¶

The Automobile dataset contains details on 205 car models from various manufacturers. It includes 24 attributes per model, covering technical specifications (engine size, horsepower, peak RPM, etc.), performance metrics (city and highway MPG), and economic data (price). This analysis aims to uncover patterns in fuel economy, pricing, and manufacturer characteristics.

Data Cleaning¶

  1. Dropped Unused Columns: Removed redundant features like normalized-losses and symboling to focus on relevant attributes.
  2. Duplicates: Verified no exact duplicate rows existed (205 → 205 entries).
  3. Missing Values:
    • Converted placeholder "?" to NaN for numeric columns (bore, stroke, horsepower, peak-rpm, price).
    • Used dropna() to discard rows missing any critical numeric data, resulting in a clean set of valid entries.
  4. Type Conversions:
    • Cast all float columns representing counts or measurements to int64 (e.g., wheel-base, engine-size, city-mpg).
    • Ensured string fields (e.g., body-style, make) were trimmed and lowercased for consistency.

Missing Data¶

  • Before cleaning, five rows contained "?" placeholders in key numeric fields.
  • These were coerced to NaN and dropped, ensuring all remaining records are complete and numeric.
  • As a result, subsequent aggregations and visualizations used only fully observed data.
In [8]:
# 1. List the object-dtype columns that really hold numbers
numeric_obj_cols = ['bore', 'stroke', 'horsepower', 'peak-rpm', 'price']

# 1. Coerce bad values into NaN
automobiles_df[numeric_obj_cols] = (
    automobiles_df[numeric_obj_cols]
      .apply(pd.to_numeric, errors='coerce')
)

# 2. (Optionally) drop rows where these are still missing
automobiles_df.dropna(subset=numeric_obj_cols, inplace=True)

# 3. Now safely cast to int64
automobiles_df[numeric_obj_cols] = automobiles_df[numeric_obj_cols].astype(np.int64)

# 4. Verify

Data Stories and Visualizations¶

  1. Price vs. Fuel Economy
    • Top 5 Most Expensive: Mercedes-Benz ($45 400), BMW ($41 315 & $36 880), Mercedes-Benz ($40 960), Porsche ($37 028) average city MPG 14–17, highway MPG 16–25.
    • Top 5 Cheapest: Subaru ($5 118), Chevrolet ($5 151), Mazda ($5 195), Toyota ($5 348), Mitsubishi ($5 389) average city MPG 31–47, highway MPG 36–53.

Insight: Luxury/performance cars sacrifice fuel economy; budget models deliver roughly double the MPG.

  1. Manufacturer Fuel Efficiency
    • Computed average (city-mpg + highway-mpg)/2 for each make.
    • Chevrolet leads with ~43 mpg on highways and ~41 mpg in the city.
    • Other efficient brands: Honda, Volkswagen, Volvo (~31–32 mpg combined).
    • Performance brands like Jaguar, Mercedes-Benz, Porsche fall below 20 mpg.
  2. Engine Capacity
    • The maximum engine size observed is 3261 cc, found in [manufacturer/model]. (Filter via df.nlargest(1,'engine-size')).
    • High-displacement models correlate with lower MPG and higher price.
    1. Model Counts by Manufacturer
    • BMW offers the most distinct models in this sample (count = 7).
    • Other prolific makes: Mercedes-Benz (6), Toyota (6), Honda (5).

  1. Inverse relationship: Fuel efficiency (whether city, highway, or average) is negatively correlated with price in this dataset.
  2. Cluster structure:
    • High-price, low-mpg cluster (luxury/performance models).
